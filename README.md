# Vacpyre

Entry for Octave Jam 2026.

Runs on New 3DS, Wii and GameCube, Linux and Windows.

Wii version requires GameCube controller.

GameCube version has no music (and may crash after multiple play throughs).

Linux version requires GLIBC 2.41.

Build with Octave bf03dade3fa8f9cd0da35e6a14c4f9adc3abf0a7 or later.

Hold Z or Select and press D-Pad Up / Down to hop between checkpoints.
